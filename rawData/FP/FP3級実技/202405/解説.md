# FP技能士試験 解説

## 問1

**正解: 1**

**分野**: ライフプランニング（FP倫理・関連法規）

**解説**:
FPは、関連業法等を順守して業務を行う必要があります。

1.  **不適切**：投資助言・代理業の登録を受けていないFPが、顧客と投資顧問契約を締結し、特定の上場株式の投資判断について有償で助言をする行為は、**金融商品取引法**に違反します。有償で個別の投資助言を行うには、投資助言・代理業の登録が必要です。
2.  **適切**：税理士の登録を受けていないFPが、有料のセミナーにおいて、仮定の事例に基づき、一般的な税法の解説を行うことは、**税理士法**に抵触しません。個別具体的な税務相談や税務書類の作成をしない限り、一般的な税法の説明は可能です。
3.  **適切**：生命保険募集人・保険仲立人の登録を受けていないFPが、変額年金保険の一般的な商品内容について説明を行うことは、**保険業法**に抵触しません。保険商品の募集行為（契約の締結を勧誘することなど）を行わない範囲での一般的な情報提供は認められています。

したがって、最も不適切なものは1です。

**キーワード**: 金融商品取引法、投資助言・代理業、税理士法、保険業法、FPの業務範囲

## 問2

**分野**: ライフプランニング

**解説**:
キャッシュフロー表の空欄(ア)～(ウ)の計算を行います。

*   **空欄(ア) 4年後の基本生活費**
    基本生活費は毎年2%ずつ増加すると記載されています。
    基準年の基本生活費：240万円
    4年後の基本生活費 = 240万円 × (1 + 0.02)^4
    = 240万円 × 1.08243216
    = 259.783...万円
    万円未満を四捨五入すると、**260万円**となります。

*   **空欄(イ) 基準年の年間収支**
    年間収支は、収入合計から支出合計を差し引いて計算します。
    基準年の収入合計：480万円
    基準年の支出合計：476万円
    基準年の年間収支 = 480万円 - 476万円 = **4万円**

*   **空欄(ウ) 2年後の金融資産残高**
    金融資産残高は、前年末の金融資産残高にその年の年間収支を加えて計算します。
    基準年末の金融資産残高：680万円
    1年後の年間収支：504万円 (収入合計) - 471万円 (支出合計) = 33万円
    2年後の年間収支：509万円 (収入合計) - 596万円 (支出合計) = -87万円

    1年年末の金融資産残高 = 基準年末の金融資産残高 + 1年後の年間収支
    = 680万円 + 33万円 = 713万円

    2年年末の金融資産残高 = 1年年末の金融資産残高 + 2年後の年間収支
    = 713万円 + (-87万円) = **626万円**

上記の計算結果と選択肢を比較します。
1.  (ア) 260万円：計算結果は260万円なので、正しいです。
2.  (イ) 33万円：計算結果は4万円なので、誤りです。
3.  (ウ) 662万円：計算結果は626万円なので、誤りです。

「誤っているものはどれか」という問いに対して、(イ)と(ウ)が誤りですが、選択肢3が解答として示されているため、(ウ)が不適切なものとして選ばれています。

**キーワード**: キャッシュフロー表、年間収支、金融資産残高、基本生活費、増減率

## 問6

**正解: 1**

**分野**: 金融資産運用

**解説**:
WA株式会社とWB株式会社の投資指標をそれぞれ計算し、記述の適切性を判断します。

**WA株式会社**
*   株価：1,200円
*   1株当たり当期純利益：200円
*   1株当たり純資産：1,500円
*   1株当たり年間配当金：25円

1.  **株価収益率(PER)** = 株価 ÷ 1株当たり当期純利益 = 1,200円 ÷ 200円 = **6倍**
2.  **株価純資産倍率(PBR)** = 株価 ÷ 1株当たり純資産 = 1,200円 ÷ 1,500円 = **0.8倍**
3.  **配当利回り** = 1株当たり年間配当金 ÷ 株価 = 25円 ÷ 1,200円 ≒ **2.08%**

**WB株式会社**
*   株価：3,000円
*   1株当たり当期純利益：300円
*   1株当たり純資産：2,000円
*   1株当たり年間配当金：50円

1.  **株価収益率(PER)** = 株価 ÷ 1株当たり当期純利益 = 3,000円 ÷ 300円 = **10倍**
2.  **株価純資産倍率(PBR)** = 株価 ÷ 1株当たり純資産 = 3,000円 ÷ 2,000円 = **1.5倍**
3.  **配当利回り** = 1株当たり年間配当金 ÷ 株価 = 50円 ÷ 3,000円 ≒ **1.67%**

各記述の検証：

1.  **株価収益率(PER)で比較した場合、株価はWA株式会社よりWB株式会社の方が割安である。**
    *   WAのPER：6倍、WBのPER：10倍。PERは低いほど割安とされます。
    *   WAの方がPERが低いため、WAの方が割安です。
    *   記述は「WA株式会社よりWB株式会社の方が割安である」とあり、これは「WBの方が割安」という意味になりますが、実際はWAの方が割安です。したがって、この記述は**不適切**です。

2.  **株価純資産倍率(PBR)で比較した場合、株価はWA株式会社よりWB株式会社の方が割高である。**
    *   WAのPBR：0.8倍、WBのPBR：1.5倍。PBRは低いほど割安とされます。
    *   WBの方がPBRが高いため、WBの方が割高です。
    *   記述は「WB株式会社の方が割高である」とあり、これは**適切**です。

3.  **配当利回りで比較した場合、収益性はWA株式会社よりWB株式会社の方が低い。**
    *   WAの配当利回り：2.08%、WBの配当利回り：1.67%。配当利回りは高いほど収益性が高いとされます。
    *   WAの方が配当利回りが高いため、WAの方が収益性が高いです。
    *   記述は「WB株式会社の方が低い」とあり、これは**適切**です。

最も不適切なものは1です。

**キーワード**: 株価収益率(PER)、株価純資産倍率(PBR)、配当利回り、割安、割高、投資指標

## 問8

**正解: 1**

**分野**: 金融資産運用

**解説**:
外貨定期預金の満期時の元利合計額を円転した金額を計算します。

1.  **満期時の米ドル建て元利合計額の計算**
    *   預入額：10,000米ドル
    *   預入期間：12ヵ月（1年）
    *   預金金利：1.50%（年率）
    *   元利合計額 = 預入額 × (1 + 預金金利)
    *   = 10,000米ドル × (1 + 0.015)
    *   = 10,000米ドル × 1.015 = **10,150米ドル**

2.  **円転時の為替レートの選択**
    *   満期時に米ドルを円に転換する（顧客が米ドルを売って円を買う）ので、銀行にとっては顧客から米ドルを買い取る形になります。
    *   この場合、銀行が米ドルを買い取るレートである**TTB (Telegraphic Transfer Buying rate)** を使用します。
    *   満期時の為替レートのTTB：124.40円/米ドル

3.  **円転後の金額の計算**
    *   円転金額 = 元利合計額(米ドル) × TTB
    *   = 10,150米ドル × 124.40円/米ドル = **1,262,660円**

したがって、満期時の外貨ベースの元利合計額を円転した金額は1,262,660円です。

**キーワード**: 外貨定期預金、為替レート、TTB (Telegraphic Transfer Buying rate)、TTS (Telegraphic Transfer Selling rate)、元利合計

## 問9

**正解: 2**

**分野**: 不動産

**解説**:
建築物の延べ面積の最高限度を計算します。

1.  **敷地面積の計算**
    *   図より、敷地は長方形であり、縦15m、横30mです。
    *   敷地面積 = 30m × 15m = **450m²**

2.  **容積率の限度の算出**
    容積率の限度は、指定容積率と前面道路幅員による容積率のうち、いずれか小さい方が適用されます。
    *   **指定容積率**: <資料>に「指定容積率 400%」と記載されています。
    *   **前面道路幅員による容積率**:
        *   前面道路の幅員：6m
        *   法定乗数：近隣商業地域なので6/10
        *   前面道路幅員による容積率 = 6m × 6/10 × 100% = 3.6m × 100% = **360%**
    *   指定容積率(400%)と前面道路幅員による容積率(360%)を比較すると、**360%**が容積率の限度となります。

3.  **延べ面積の最高限度の計算**
    *   延べ面積の最高限度 = 敷地面積 × 容積率の限度
    *   = 450m² × 360% (3.6) = **1,620m²**

したがって、延べ面積の最高限度は1,620m²です。

## 問13

**正解: 2**

**分野**: リスク管理（生命保険と税金）

**解説**:
死亡保険金にかかる税金は、保険契約者、被保険者、保険金受取人の関係によって異なります。

*   **相続税**：契約者＝被保険者≠受取人 の場合
*   **所得税（一時所得）**：契約者≠被保険者＝受取人 の場合
*   **贈与税**：契約者≠被保険者≠受取人 の場合

各選択肢の課税関係を検証します。

1.  **保険契約者(保険料負担者)および被保険者が大久保さん、死亡保険金受取人が大久保さんの妻の場合**
    *   契約者(夫) = 被保険者(夫) ≠ 受取人(妻) の関係です。
    *   この場合、死亡保険金は**相続税**の課税対象となります。
    *   記述は「贈与税の課税対象となる」としているため、**不適切**です。

2.  **保険契約者(保険料負担者)が大久保さん、被保険者が大久保さんの妻、死亡保険金受取人が大久保さんの場合**
    *   契約者(夫) ≠ 被保険者(妻) = 受取人(夫) の関係です。
    *   この場合、死亡保険金は一時所得として**所得税・住民税**の課税対象となります。
