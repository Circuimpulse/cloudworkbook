# FP技能士試験 解説

## 問1

**正解: 1**

**分野**: ライフプランニング (FP倫理・関連法規)

**解説**:
FPが顧客に対して行う行為は、関連法規によって厳しく制限されています。
1. **正解の理由**: 金融商品取引業の登録を受けていないFPが、顧客と投資顧問契約を締結し、株式の個別銘柄の購入を勧める行為は、金融商品取引法に抵触します。投資顧問契約の締結や個別銘柄の購入勧誘は、金融商品取引業者が行うべき業務です。したがって、これは最も不適切な記述です。
2. 弁護士の登録を受けていないFPが、顧客の求めに応じ、有償で任意後見契約の受任者となることは、任意後見契約の受任者に資格制限がないため、弁護士法に抵触しません。任意後見契約は、本人の判断能力が低下した場合に備えて、あらかじめ本人（委任者）と任意後見人となる者（受任者）が、公正証書で締結する契約です。
3. 税理士の登録を受けていないFPが、顧客から配偶者控除と配偶者特別控除の適用要件を聞かれ、所得税法の条文等を示しつつ、それぞれの適用要件の違いを説明することは、税理士法に抵触しません。これは一般的な税法の情報提供にとどまり、個別具体的な税務相談や税務書類の作成には該当しないためです。
4. 生命保険募集人の登録を受けていないFPが、顧客からライフプランの相談を受け、老後資金準備のための生命保険の一般的な活用方法を説明することは、保険業法に抵触しません。これは一般的な保険商品の情報提供やライフプランニングに関する助言であり、特定の保険商品の勧誘・販売行為には該当しないためです。

**キーワード**: 金融商品取引法、弁護士法、税理士法、保険業法、投資顧問契約、任意後見契約、配偶者控除、配偶者特別控除、生命保険募集人

## 問2

**正解: 3**

**分野**: ライフプランニング (資金計画係数)

**解説**:
各種係数は、将来の資金計画を立てる上で重要なツールです。
1. 一定の利率で複利運用しながら一定期間経過後の元利合計額を試算する際、現在保有する資金の額に乗じる係数は、終価係数です。終価係数は、現在の資金が将来いくらになるかを計算する際に使用します。これは正しい記述です。
2. 一定の利率で複利運用しながら一定期間、毎年一定金額を積み立てた場合の一定期間経過後の元利合計額を試算する際、毎年の積立額に乗じる係数は、年金終価係数です。年金終価係数は、毎年一定額を積み立てた場合の将来の合計額を計算する際に使用します。これは正しい記述です。
3. **正解の理由**: 一定の利率で複利運用しながら一定期間、毎年一定金額を受け取るために必要な元本を試算する際、毎年受け取りたい金額に乗じる係数は、年金現価係数です。資本回収係数は、現在の一定金額を元本として、一定期間にわたって毎年均等に分割して受け取る場合の毎年の受取額を計算する際に使用する係数です。したがって、この記述は不適切です。
4. 一定の利率で複利運用しながら一定期間経過後に目標とする額を得るために必要な毎年の積立額を試算する際、目標とする額に乗じる係数は、減債基金係数です。減債基金係数は、将来の目標額を達成するために毎年いくら積み立てる必要があるかを計算する際に使用します。これは正しい記述です。

**キーワード**: 終価係数、年金終価係数、年金現価係数、資本回収係数、減債基金係数、複利運用、元利合計額、元本

**解説**:
雇用保険の失業等給付に関する知識は、生活設計において重要です。
1. 雇用保険の一般被保険者が失業した場合、基本手当を受給するためには、原則として、離職の日以前2年間に被保険者期間が通算して12ヵ月以上あること等の要件を満たす必要があります。これは正しい記述です。
2. **正解の理由**: 正当な理由がなく自己都合により退職し、基本手当の受給を申請した場合、7日間の待期期間経過後、給付制限期間として基本手当を受給できない期間があります。2020年10月1日以降の離職からは、この給付制限期間は原則2ヵ月間となっています。以前は3ヵ月間でしたが、2020年の法改正により短縮されました。したがって、「4ヵ月間」としているこの記述は不適切です。
3. 基本手当の受給期間内に、出産、疾病等の理由で引き続き30日以上職業に就くことができない場合、所定の申出により、受給期間を離職日の翌日から最長4年まで延長することができます。これは正しい記述です。
4. 雇用保険の高年齢被保険者が失業した場合、高年齢求職者給付金を受給するためには、原則として、離職の日以前1年間に被保険者期間が通算して6ヵ月以上あること等の要件を満たす必要があります。これは正しい記述です。
