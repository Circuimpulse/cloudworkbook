# FP技能士試験 解説

## 問1

**正解: 1**

**分野**: ライフプランニング（FP倫理・関連法規）

**解説**:
FPは、関連法規（弁護士法、税理士法、金融商品取引法、社会保険労務士法など）に抵触しない範囲で業務を行う必要があります。

1.  **正解の理由**: 社会保険労務士の登録を受けていないFPであっても、公的年金の老齢給付の試算や説明は、FP業務の範囲内とされています。顧客の年金受給額の見込額を試算し、説明することは問題ありません。
2.  **不正解の理由**: 税理士の登録を受けていないFPが、顧客の年間収入や家族構成をもとに寄附金控除の額を計算し、確定申告書の作成を代行することは、税理士法に違反する「税務代行」にあたります。税務相談は可能ですが、具体的な計算や申告書作成の代行はできません。
3.  **不正解の理由**: 金融商品取引業者の登録を受けていないFPが、顧客が選んだ銘柄の株価チャートを示しながら「投資のタイミングを有償で助言」することは、金融商品取引法に違反する「投資助言業」にあたります。一般的な金融商品の説明や情報提供は可能ですが、個別の銘柄について投資判断をさせるような具体的な助言はできません。
4.  **不正解の理由**: 弁護士の登録を受けていないFPが、顧客と代理人契約を締結し、顧客の代理人として他の相続人との遺産分割協議を行うことは、弁護士法に違反する「法律事務」にあたります。遺産分割に関する一般的な情報提供は可能ですが、代理人として交渉することはできません。

**キーワード**: FPの関連法規、税理士法、金融商品取引法、弁護士法、社会保険労務士法

## 問2

**正解: 4**

**分野**: ライフプランニング（公的医療保険）

**解説**:
全国健康保険協会管掌健康保険（協会けんぽ）に関する知識が問われています。

1.  **適切な記述**: 協会けんぽの一般保険料率は、医療費の地域差を反映して都道府県ごとに定められています。一方、介護保険料率は、全国一律に定められています（40歳以上65歳未満の被保険者に適用）。
2.  **適切な記述**: 高額療養費制度は、医療機関に支払った医療費の自己負担額が一定額を超えた場合にその超えた部分が支給される制度です。しかし、保険診療の対象とならない差額ベッド代（特別室料）や入院時の食事代、生活療養費に係る費用などは高額療養費の支給対象にはなりません。
3.  **適切な記述**: 協会けんぽの被保険者が退職により資格を喪失した場合でも、所定の要件（被保険者期間が継続して2ヵ月以上など）を満たせば、退職日の翌日から最長で2年間、任意継続被保険者となることができます。
4.  **不適切な記述**: 夫婦がともに被保険者である場合において、妻が出産したとき、出産育児一時金は「被保険者である妻」に対して支給されます。家族出産育児一時金は、被保険者の「被扶養者」が出産した場合に被保険者に支給されるものです。したがって、被保険者である妻の出産では、家族出産育児一時金は支給されません。

**キーワード**: 協会けんぽ、一般保険料率、介護保険料率、高額療養費、任意継続被保険者、出産育児一時金、家族出産育児一時金

## 問3

**正解: 1**

**分野**: ライフプランニング（雇用保険）

**解説**:
雇用保険に関する知識が問われています。

1.  **正解の理由**: 65歳以上の労働者でも、雇用保険の被保険者となることができます。複数の事業所に雇用される場合は、各事業所での1週間の所定労働時間が20時間未満であっても、2つの事業所での所定労働時間の合計が20時間以上であり、それぞれの事業所で雇用保険の適用がある場合は、申出により、高年齢被保険者となることができます（高年齢二以上事業所勤務者）。
2.  **不正解の理由**: 雇用保険の保険料のうち、「失業等給付・育児休業給付に係る保険料」は、事業主と労働者がそれぞれ負担する（折半ではないが、一定の割合で負担する）のに対し、「雇用保険二事業に係る保険料」は、事業主が全額を負担します。
3.  **不正解の理由**: 特定受給資格者等を除く一般の受給資格者に支給される基本手当の所定給付日数は、算定基礎期間（被保険者期間）によって異なります。算定基礎期間が「20年以上」の場合、給付日数は240日です（10年以上20年未満の場合は150日、1年未満10年未満の場合は90日）。120日は、算定基礎期間が1年以上10年未満で離職日において30歳以上65歳未満の場合などに該当します。
4.  **不正解の理由**: 育児休業給付金の額は、育児休業開始から180日目までは休業開始時賃金日額に支給日数を乗じた額の100分の67に相当する額となります。181日目以降は100分の50に相当する額です。100分の40は誤りです。

**キーワード**: 雇用保険、高年齢被保険者、失業等給付、育児休業給付、雇用保険二事業、基本手当、所定給付日数、育児休業給付金

## 問5

**正解: 3**

**分野**: ライフプランニング（企業年金・退職給付）

**解説**:
確定拠出年金の企業型年金導入と既存制度との関係についての知識が問われています。

1.  **不正解の理由**: 厚生年金適用事業所の事業主が企業型年金を実施する場合、労使合意に基づいて規約を作成し、当該規約について厚生労働大臣の承認を受ける必要があります。都道府県知事ではありません。
2.  **不正解の理由**: 退職給与規程に基づく退職一時金制度を実施している企業が、同制度を廃止して確定拠出年金に資産を移換する場合、その資産は「複数年度にわたって」移換することとされており、単年度で一括して移換することはできません。
3.  **正解の理由**: 確定給付企業年金を実施している企業が、同制度に係る資産を企業型年金に移換した場合、確定給付企業年金の加入期間は、60歳に達した日の前日が属する月以前の期間に限り、確定拠出年金の通算加入者等期間に通算されます。これにより、ポータビリティが確保されます。
4.  **不正解の理由**: 中小企業退職金共済（中退共）に加入している企業が、事業の拡充等により中小企業者でなくなり、所定の申出により共済契約を解除する場合、当該契約に係る資産（掛金）は、企業型年金に移換することができます。これは中退共の「特定事業主掛金」制度を利用して行われます。

**キーワード**: 確定拠出年金（企業型）、規約承認、退職一時金、確定給付企業年金、通算加入者等期間、中小企業退職金共済（中退共）、ポータビリティ

**解説**:

1.  **不正解の理由**: 死亡保障を目的とした少額短期保険の保険料は、所得税の生命保険料控除の対象となりません。生命保険料控除の対象となるのは、一般の生命保険料、介護医療保険料、個人年金保険料です。

## 問12

**正解: 2**

**分野**: リスク管理（民間保険の基礎知識）

**解説**:
生命保険の一般的な商品性に関する知識が問われています。

1.  **不正解の理由**: 低解約返戻金型終身保険は、保険料払込期間中の解約返戻金が通常の終身保険に比べて低く抑えられているため、その分、保険料が割安に設定されています。しかし、死亡保険金額は他の契約条件が同一の終身保険と同額であり、低く抑えられているわけではありません。
2.  **正解の理由**: 変額保険（終身型）では、運用実績によって保険金額や解約返戻金が変動しますが、死亡保険金については、契約時に定めた「基本保険金額（最低保証額）」が保証されており、運用実績にかかわらずこれを下回ることはありません（ただし、災害死亡の場合は別途特約などによる保障がある場合が多い）。
3.  **不正解の理由**: 定期保険特約付終身保険（更新型）において、定期保険特約を同額の保険金額で更新する場合、原則として、被保険者の健康状態についての告知や医師の診査は不要です。健康状態にかかわらず更新できるのが、更新型定期保険特約の特徴です。ただし、保険料は更新時の年齢に応じて再計算されるため、通常は高くなります。
4.  **不正解の理由**: 逓減定期保険は、保険期間の経過に伴い「保険金額」が所定の割合で逓減していく保険です。保険料は一定の場合と、逓減していく場合とがあります。設問では保険金額が一定とあるため、誤りです。
