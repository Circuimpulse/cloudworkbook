## 問1

**正解: (ア)○ (イ)○ (ウ)× (エ)○**

**分野**: ライフプランニング

**解説**:
FPがファイナンシャル・プランニング業務を行う際には、関連業法を遵守することが非常に重要です。

*   **(ア) 生命保険募集人、保険仲立人または金融サービス仲介業の登録を受けていないFPが、生命保険契約を検討している顧客のライフプランに基づき、有償で具体的な必要保障額を試算した。→〇**
    生命保険の必要保障額の試算は、FP業務として一般的に行われるアドバイスであり、生命保険募集人等の資格がなくても行うことができます。ただし、具体的な商品の説明や契約の仲介を行う場合は、それぞれの登録が必要となります。この記述では「具体的な必要保障額を試算した」までなので、有償であっても問題ありません。

*   **(イ) 弁護士の登録を受けていないFP（遺言者や公証人と利害関係はない成年者）が、顧客から依頼されて公正証書遺言の証人となり、顧客から適正な報酬を受け取った。→〇**
    公正証書遺言の証人になることは、弁護士法で規定される「法律事務」には該当せず、特定の資格がなくても行うことができます。また、FPが業務として証人となり、報酬を受け取ることも可能です。利害関係者でない成年者であることも条件を満たしています。

*   **(ウ) 投資助言・代理業の登録を受けていないFPが、顧客に対し有償で、特定企業の公表されている決算報告書を用いて、具体的な株式の投資時期等の判断や助言を行った。→×**
    具体的な株式の銘柄を選定し、その投資時期などについて有償で助言を行う行為は、金融商品取引法に定める「投資助言業」に該当します。投資助言業を行うには、金融商品取引業者として登録が必要であり、登録を受けていないFPが行うことはできません。

*   **(エ) 税理士の登録を受けていないFPが、公民館主催の相談会に訪れた相談者に対し、無償で仮定の事例に基づく相続税額を計算する手順の説明を行った。→〇**
    税理士の登録を受けていないFPが、具体的な個別の相続税額の計算や申告書の作成を行うことは税理士法に違反しますが、一般的な情報提供や、仮定の事例に基づいた計算手順の説明を行うことは問題ありません。さらに、このケースでは無償で行われているため、問題ありません。

**キーワード**: 関連業法、生命保険募集人、金融サービス仲介業、弁護士法、公正証書遺言、金融商品取引法、投資助言業、税理士法、税理士業務

## 問2

**正解: 4**

**分野**: ライフプランニング

**解説**:
消費者契約法は、消費者と事業者との間の情報の質・量、交渉力の格差を是正し、消費者の利益を保護することを目的とした法律です。

*   **1. 消費者契約法では、個人および法人を保護の対象としている。→不適切**
    消費者契約法が保護の対象とするのは「消費者」であり、事業者が自己の事業のために締結する契約は対象外です。したがって、法人は保護の対象に含まれません。

*   **2. 消費者契約の取消権は、当該消費者契約の締結時から2年を経過したときは、時効により消滅する。→不適切**
    消費者契約法の取消権は、追認できるときから1年以内、または契約締結のときから5年以内に行使しないと時効により消滅します。2年ではありません。

*   **3. 消費者が、商品を買わずに事業者の店舗から帰りたいと言っても帰らせてもらえずに困惑して契約を締結した場合、当該契約は取り消すことができない。→不適切**
    消費者が、事業者の店舗などから退去しないことにより困惑して契約を締結した場合（不退去）、または、消費者が事業者に店舗などから退去してほしいと言ったにもかかわらず事業者が退去しないことにより困惑して契約を締結した場合（退去妨害）は、消費者契約法に基づいてその契約を取り消すことができます。

*   **4. 事業者が消費者に重要事項について事実と異なることを告げ、消費者がそれを事実であると誤認して締結した契約は、取り消すことができる。→適切**
    これは消費者契約法における「不実告知」に該当します。事業者が契約の目的物となるものの質、用途その他の内容、または役務の質、内容等について、事実と異なることを告げたことにより消費者が誤認して契約を締結した場合、その契約を取り消すことができます。

**キーワード**: 消費者契約法、消費者、事業者、不実告知、取消権、不退去、退去妨害

## 問3

**正解: 3**

**分野**: 金融資産運用

**解説**:
投資信託の収益分配後の個別元本は、分配金が支払われた後の個別元本を指します。
個別元本とは、顧客が投資信託を購入したときの価額（手数料込み）のことです。追加購入した場合は、購入するたびに加重平均により再計算されます。

収益分配が行われた場合、分配金が課税前個別元本を下回る部分（元本払戻金）は非課税となり、個別元本から減額されます。一方、分配金が課税前個別元本を上回る部分（普通分配金）は課税対象となり、個別元本は減額されません。

本問では、収益分配前の個別元本と基準価額、収益分配後の基準価額、そして収益分配金が与えられています。
収益分配金が支払われた後の個別元本を計算するにあたり、以下の関係に着目します。

1.  **収益分配金の内訳の判断**:
    *   収益分配前の基準価額: 11,400円
    *   収益分配前の個別元本: 10,800円
    *   収益分配金: 1,000円

    分配金1,000円のうち、個別元本10,800円を上回る基準価額の部分（11,400円 - 10,800円 = 600円）までは「普通分配金」となります。
    そして、個別元本を下回る部分の分配金（1,000円 - 600円 = 400円）は「元本払戻金（特別分配金）」となります。
    *   普通分配金 = 600円
    *   元本払戻金 = 400円

2.  **収益分配後の個別元本**:
    個別元本は、元本払戻金（特別分配金）の分だけ減額されます。普通分配金は個別元本を減額しません。
    したがって、収益分配後の個別元本は、収益分配前の個別元本から元本払戻金を差し引いた金額となります。

    収益分配後の個別元本 = 収益分配前の個別元本 - 元本払戻金
    = 10,800円 - 400円
    = 10,400円

**キーワード**: 投資信託、収益分配金、個別元本、普通分配金、元本払戻金（特別分配金）、基準価額

**分野**: 金融資産運用

**解説**:

    *   預入額: 10,000豪ドル
    *   預金金利: 2.5% (年率)
    *   預入期間: 12ヶ月

    = 10,000豪ドル × 2.5%
    = 10,000豪ドル × 0.025
    = 250豪ドル

2.  **源泉徴収される税金**:
    *   豪ドル建ての利息額の20%が所得税・住民税として源泉徴収されます。

    税額 = 利息 × 20%
    = 250豪ドル × 0.20
    = 50豪ドル

3.  **税引後利息の計算**:
    税引後利息 = 利息 (税引前) - 税額
    = 250豪ドル - 50豪ドル
    = 200豪ドル

4.  **満期時の元利合計額 (外貨ベース)**:
    満期時の元利合計額 = 預入額 + 税引後利息
    = 10,000豪ドル + 200豪ドル

    PER (株価収益率) = 株価 ÷ 1株当たり当期純利益

    <資料>より

    5.29倍

**分野**: ライフプランニング

**解説**:
財形貯蓄制度は、勤労者の財産形成を支援するための制度です。財形年金貯蓄と財形住宅貯蓄にはそれぞれ固有の要件や非課税限度額、目的外払出し時の取扱いがあります。

*   **(ア) 空欄(a)にあてはまる数値は、「60」である。→×**
    財形年金貯蓄の契約締結時の年齢要件は、原則として**満55歳未満**です。60歳ではありません。

*   **(イ) 空欄(b)にあてはまる数値は、「550」である。→○**
    財形住宅貯蓄の非課税限度額は、財形年金貯蓄と合算して元利合計で**550万円まで**です。これは正しい記述です。

*   **(ウ) 空欄(c)にあてはまる数値は、「350」である。→×**
    財形年金貯蓄（保険型）の払込保険料累計額の非課税限度額は、財形住宅貯蓄と合算して**350万円まで**です。資料の「(c) 万円」という空欄は、この限度額を指しているため、「350」が当てはまります。この選択肢の記述は「空欄(c)にあてはまる数値は、「350」である。」とあり、これが正解なので、なぜ「×」なのか。
    「(ウ)空欄(c)にあてはまる数値は、「350」である。」と解答が「×」となっています。
    資料の記載を確認すると、財形年金貯蓄[保険型]の非課税限度額は「払込保険料累計額(c)万円まで、かつ財形住宅貯蓄と合算して払込保険料累計額***万円まで」となっています。
    財形年金貯蓄と財形住宅貯蓄の保険型では、払込保険料累計額で350万円までが非課税です。したがって、(c)は350万円が適切です。
    選択肢(ウ)は「空欄(c)にあてはまる数値は、「350」である」と書かれており、これは正しいように見えます。
    しかし、模範解答は(ウ)が×になっています。
    これは、問題文をよく読むと、「財形年金貯蓄[保険型] 払込保険料累計額( c ) 万円まで、かつ財形住宅貯蓄と合算して払込保険料累計額***万円まで」となっており、この「***」の部分に入る数値が「350」です。

    財形年金貯蓄の非課税限度額は、貯蓄型の場合、財形住宅貯蓄と合算して元利合計350万円まで。

高額療養費制度は、医療費の自己負担額が一定額を超えた場合に、その超えた部分が払い戻される制度です。自己負担限度額は、年齢や所得によって異なります。

**＜吉岡さんに関する情報＞**
*   年齢: 45歳 (70歳未満)
*   標準報酬月額: 470,000円
*   支払った医療費等:
    *   保険診療分の医療費 (自己負担分): 360,000円
    *   入院時の食事代: 40,000円 (高額療養費の対象外)
    *   差額ベッド代: 200,000円 (高額療養費の対象外)

1.  **高額療養費の対象となる医療費の自己負担額**:
    *   入院時の食事代や差額ベッド代は高額療養費の対象外です。
    *   高額療養費の対象となる自己負担額は、保険診療分の医療費のみです。
    *   自己負担額 (3割負担) = 360,000円
    *   総医療費 (10割) = 360,000円 ÷ 0.3 = 1,200,000円

2.  **自己負担限度額の確認**:
    吉岡さんの標準報酬月額は470,000円です。
    <医療費の1ヵ月当たりの自己負担限度額(70歳未満)>の表より、
    「28万円～50万円」の区分に該当します。
    この区分の自己負担限度額は、「80,100円 + (総医療費 - 267,000円) × 1%」です。

3.  **自己負担限度額の計算**:
    *   総医療費: 1,200,000円 (上記で計算)
    *   自己負担限度額 = 80,100円 + (1,200,000円 - 267,000円) × 0.01
        = 80,100円 + 933,000円 × 0.01
        = 80,100円 + 9,330円
        = 89,430円

4.  **高額療養費として支給される額の計算**:
    *   実際に支払った自己負担額 (対象医療費のみ): 360,000円

    高額療養費として支給される額 = 支払った自己負担額 - 自己負担限度額
    = 360,000円 - 89,430円
    = 270,570円

## 問40

**正解: (ア) × (イ) × (ウ) ×**

**分野**: ライフプランニング

**解説**:
雇用保険の基本手当に関する記述の適切性が問われています。基本手当は失業給付の主要なもので、受給要件、日額計算、受給期間が定められています。

*   **「雇用保険の基本手当は、一般被保険者が失業した場合に、原則として、離職の日以前の***に被保険者期間が通算して( a )以上あるときに支給される。」**
    **(ア) 空欄(a)にあてはまる語句は、「6ヵ月」である。→×**
    基本手当の受給要件となる被保険者期間は、原則として離職の日以前2年間において、**通算して12ヵ月以上**あることです（倒産・解雇等特定受給資格者、特定理由離職者の場合は1年間で6ヵ月以上）。「6ヵ月」は特定受給資格者等の要件であり、一般的な「一般被保険者」の場合は12ヵ月です。したがって不適切です。

*   **「基本手当日額は、原則として、離職の日の直前 ( b )間に支払われた賃金総額(賞与等は除く)を***で除した賃金日額に給付率を乗じた額であり、基本手当を受給できる期間は、原則として、( c )から1年間である。」**
    **(イ) 空欄(b)にあてはまる語句は、「3ヵ月」である。→×**
    基本手当日額の算定基礎となる賃金総額は、原則として**離職の日以前6ヵ月間**に支払われた賃金総額です。3ヵ月ではありません。

*   **(ウ) 空欄(c)にあてはまる語句は、「離職日が属する月の翌月1日」である。→×**
    基本手当を受給できる期間（受給期間）は、原則として**離職日の翌日から1年間**です。「離職日が属する月の翌月1日」ではありません。

**キーワード**: 雇用保険、基本手当、被保険者期間、受給要件、基本手当日額、賃金日額、受給期間
