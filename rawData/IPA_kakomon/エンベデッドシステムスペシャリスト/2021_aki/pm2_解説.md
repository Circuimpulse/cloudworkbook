## 問題の概要

#### (2) LIDARの水平方向の角度分解能を0.1度にした。このときの回転速度を、単位(回転/秒)で答えよ。答えは小数第1位を四捨五入して、整数で求めよ。
> **正解**: 10

**思考プロセスと解説**:
1.  **LIDARの性能諸元確認**: 問題文の「〔LIDAR による計測処理〕」と「LIDAR」の項目（表4）を確認します。
    *   LIDARは「32本のレーザ光」を使用。
    *   「遮蔽物までの距離を1秒間に合計 **1,152,000ポイント**計測する。」
    *   「LIDARは、**5～20回転/秒**で水平回転することで、水平方向の角度分解能を調整する。」
2.  **角度分解能の定義**: 水平方向の角度分解能は、1回転（360度）あたりに計測するポイント数によって決まります。
    *   角度分解能 (度) = 360度 / (1回転あたりの計測ポイント数)
3.  **1回転あたりの計測ポイント数の算出**: 設問では角度分解能を0.1度としたいので、1回転に必要なポイント数は $360 \text{度} / 0.1 \text{度/ポイント} = 3,600$ ポイント/回転。
4.  **回転速度の算出**:
    *   LIDARは1秒間に合計1,152,000ポイントを計測します。このポイント数は、32本のレーザ光をすべて合わせてのもの。
    *   1本のレーザ光が1秒間に計測するポイント数: $1,152,000 \text{ポイント/秒} / 32 \text{本} = 36,000$ ポイント/秒/本。
    *   1本のレーザ光が1回転で3,600ポイントを計測すると考えると、回転速度は、1秒間に計測できるポイント数から1回転に必要なポイント数を割ることで求められます。
    *   回転速度 (回転/秒) = (1本あたり1秒間の計測ポイント数) / (1本あたり1回転の計測ポイント数)
    *   回転速度 = $36,000 \text{ポイント/秒} / 3,600 \text{ポイント/回転} = 10$ 回転/秒。
5.  **範囲の確認**: 算出した回転速度10回転/秒は、「5～20回転/秒」というLIDARの仕様範囲内に収まっています。
