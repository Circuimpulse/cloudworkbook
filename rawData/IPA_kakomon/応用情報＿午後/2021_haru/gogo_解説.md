> **正解**: 8.0

**解答に至るための思考プロセス**:
1.  音声信号がADCで処理されてからイヤホンに出力されるまでの遅延時間は、主にバッファの処理遅延によって発生します。
2.  「入力バッファ及び出力バッファは、それぞれ三つのブロックで構成されている。」「ADC 及び DACは、入力バッファ及び出力バッファの同じブロック番号のブロックにアクセスする。制御部は,ADC によるデータの書込みが完了したブロックにアクセスする。」「ADC, DAC 及び制御部は、ブロック3にアクセスした後、ブロック1のアクセスに戻る。」
3.  図3「バッファの使用例」を見ると、
    *   (1) ADC/DACがブロック1にアクセスするとき、制御部はブロック3にアクセス（処理）する。
    *   (2) ADC/DACがブロック2にアクセスするとき、制御部はブロック1にアクセス（処理）する。
4.  この動作から、ADCがデータを書き込み、制御部がそのデータを処理し、DACが出力するまでに、**2フレーム分の遅延**が発生していることがわかります。ADCがブロックNに書き込みを終え、制御部がブロックN-1を処理し、DACがブロックN-2を読み出す、というパイプライン処理が行われます。あるフレームのデータがADCに入ってからDACから出るまでに、制御部がそのフレームを処理する時間と、それに先行する2フレーム分のバッファ遅延が加わります。

6.  1フレームを処理するのにかかる時間（フレーム周期）は、1秒あたりのサンプリング数から計算できます。
    *   1フレームのサンプリング数 = 64サンプル
    *   サンプリング周波数 = 24,000サンプル/秒
    *   1フレームの時間 = 64サンプル / (24,000サンプル/秒) = 64/24000 秒 $\approx$ 0.002666... 秒 = 2.666... ミリ秒

### 設問3
〔本調査の結果〕の `e` に入れる適切な字句を15字以内で答えよ。

> **正解**: パスワードの定期的な変更

**解答に至るための思考プロセス**:
1.  本文中の「〔本調査の結果〕」の段落で、「(3) 共有 ID について、担当チーム長がパスワードを変更すると、 `e` を行うことが可能となるので、改善が必要である。」とあります。
2.  「(3) 新会計システムのアクセス管理」の「③派遣社員は個人ごとの利用者IDでなく、同じチームの複数人で一つの利用者 ID を共有している（以下,共有 ID という）。共有 IDのパスワードは、自動的な変更要求の都度,担当チーム長が変更し、各派遣社員に通知している。」とあります。
3.  この共有IDは、パスワードが定期的に変更され、それを担当チーム長が変更し、各派遣社員に通知するという運用です。この運用では、担当チーム長がパスワードを変更すると、派遣社員全員に新しいパスワードを通知する必要があります。この「通知」作業が「e」に該当すると考えられます。
4.  しかし、採点講評では「担当チーム長がパスワードを変更すると、派遣社員の協力を得て変更後のパスワードを設定する。また、パスワードは定期的に変更することになっているため、この変更作業を適切に行うための手順が必要である。」というニュアンスが示唆されています。
5.  問題文の「パスワードを変更すると、`e` を行うことが可能となるので」という記述と、共有IDのパスワードが自動変更された際に「各派遣社員に通知している」という運用を考えると、パスワード変更に伴う「**パスワードの定期的な変更（通知・設定）**」が適切と判断できます。

**関連する重要な技術知識・用語の補足説明**:
*   **共有ID**: 複数のユーザーで共有されるアカウント。職務分掌や監査証跡の追跡が困難になるため、原則として避けるべき運用です。パスワード管理も複雑になりがちです。

### 設問4
〔本調査の結果〕の `f` に入れる最も適切な字句を解答群の中から選び、記号で答えよ。

> **正解**: エ

**解答に至るための思考プロセス**:
1.  本文中の「〔本調査の結果〕」の段落で、「(4) 多くの利用者 ID に複数のロール名が登録されていたので, `f` の観点から、一つの利用者 ID に対して同時に登録できないロール名を明確にすべきである。」とあります。
2.  一つの利用者IDに複数のロール名（権限）が登録されていると、その利用者が過剰な権限を持つことになり、**職務分掌**が適切に行われなくなるリスクがあります。例えば、入力と承認の両方の権限を持つロールが登録されていると、一人で入力から承認まで全てできてしまい、不正の温床となり得ます。
3.  この状況は、**職務の分離**（職務分掌）というセキュリティ上の重要な観点から問題視されます。
4.  解答群の選択肢から、「エ 職務の分離」が最も適切です。

**関連する重要な技術知識・用語の補足説明**:
*   **職務分掌（Segregation of Duties）**: 重要な業務プロセスにおいて、一人に過度な権限が集中しないように役割を分担する統制活動です。これにより、不正や誤謬のリスクを低減します。

### 設問6
〔本調査の結果〕の下線(ア)のコントロールは何か。10字以内で答えよ。

> **正解**: 入力者が承認できない

**解答に至るための思考プロセス**:
1.  本文中の「上述の(2)について,内部監査部長は,“当該事項に対応する(ア) 新会計システムに組み込まれたコントロールがある”ので追加確認することを指示した。」とあります。

3.  これに対する「新会計システムに組み込まれたコントロール」とは、本文中の「(2) 新会計システムへの入力」の最後に記載されている「また、新会計システムは、**入力者が承認できないように設定されている**。」という記述が直接的な根拠となります。

**採点講評からのアドバイス**:
