# SectionCard

## 対象ソースファイル

`src/frontend/components/features/section-card.tsx`

## 責務・概要

セクションの進捗付きカード。学習の進捗状況（正解数/出題数、正答率）をプログレスバーで可視化する。

## Props

| Prop       | 型                | 必須 | 説明           |
| ---------- | ----------------- | ---- | -------------- |
| `section`  | `Section`         | ✅   | セクション情報 |
| `progress` | `SectionProgress` | —    | セクション進捗 |

## 表示パターン

### 未学習時

- アイコン + セクションタイトル
- 「7問の問題に挑戦しましょう」
- 「学習を始める」ボタン

### 学習中

- アイコン + セクションタイトル + 完了チェック (全問正解時: 緑チェック)
- プログレスバー + 正解数/出題数 + 正答率%
- 「学習を続ける」ボタン

## 遷移先

`/sections/{section.id}`
