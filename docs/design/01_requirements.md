# 01. 要件定義書 (Requirements Definition)

## 1. システム概要

**CloudWorkbook** は、IPA資格試験（応用情報技術者試験 午前/午後、高度試験）およびFP技能検定の過去問演習Webアプリケーションである。
スマートフォン1台で過去問演習が完結することを最優先目標とし、AI採点による記述式問題の自動フィードバック機能を備える。

## 2. 目的と目標

- **目的**: スマホ1台で資格試験の過去問演習（選択式・記述式）を効率的に行えるようにし、学習者の合格をサポートする
- **目標**:
  - 隙間時間を活用した小分けセクション学習（5〜7問1セット）
  - IPA午後試験の記述式問題にAI採点でフィードバックを提供
  - 学習履歴・正答率の可視化によるモチベーション維持と弱点克服支援
  - 本番を想定した模擬テスト（50問）による実力測定

## 3. ターゲットユーザー

- 応用情報技術者試験の受験予定者（午前4択 + 午後記述式）
- IPA高度試験の受験予定者（ネットワーク、データベース、セキュリティ等）
- FP3級・FP2級の受験予定者（学科・実技）
- IT分野の知識を体系的に身につけたいエンジニア・学生

## 4. 機能要件

### 4.1. ユーザー機能

1. **認証・ユーザー管理**
   - Clerk による新規登録、ログイン、ログアウト
   - コースセクション以降のページはログイン必須（middleware.tsで制御）
   - APIキー設定: ユーザーごとのGemini APIキーをClerk privateMetadataに保存

2. **セクション学習（一問一答）**
   - 試験区分別のセクション選択・実施（5〜7問1セット）
   - 対応する問題形式:
     - `choice`: 4択問題（応用情報午前、FP学科等）
     - `true_false`: ○×問題
     - `fill_in`: 穴埋め問題（FP実技の計算問題等）
     - `select`: 語群選択問題
     - `descriptive`: 自由記述問題（IPA午後等）
   - 即時フィードバック（正解・解説の表示）
   - お気に入り（3段階ブックマーク）

3. **AI採点機能（記述式問題）**
   - Gemini APIを利用した記述式回答の自動採点
   - スコア（0〜100）とフィードバック解説の表示
   - ユーザー自身のAPIキーで実行（Clerk privateMetadata経由）
   - 「回答してAI採点」（自動実行）と「模範解答だけ見る」の2つのフロー

4. **模擬テスト機能**
   - 全範囲からランダムに50問を出題
   - 終了後のスコア算出と結果表示

5. **学習履歴・分析機能**
   - 間違えた問題の復習機能
   - お気に入り問題のフィルタリング復習（3段階 x OR/AND）
   - セクション別の進捗管理

### 4.2. システム・管理者機能

1. **問題データインポート**
   - CSV/Markdownからの過去問インポートスクリプト
   - PDF→Markdown変換（Gemini APIで自動抽出）
   - 対応試験: 応用情報午前/午後、FP3級学科/実技、FP2級実技、IPA高度試験（SC, NW, DB等）

## 5. 非機能要件

### 5.1. パフォーマンス
- N+1クエリを排除した効率的なDBアクセス（DBインデックス5個追加済み）
- Vercel Edge Networkによる配信

### 5.2. デバイス・UI/UX
- スマートフォンファースト設計（横スクロール排除、文字サイズ最適化）
- レスポンシブ対応（スマホ・タブレット・PC）
- Markdown問題文のレンダリング（画像・表・コードブロック対応）

### 5.3. セキュリティ
- 認証はClerkに委譲（パスワード自社管理なし）
- APIバリデーション: Zodスキーマによる入力検証
- XSS対策: HTMLタグ・onイベント属性・javascript:プロトコルの除去
- 記述式回答: 500字制限（フロント + API両方で制限）
- APIキーはClerk privateMetadataに保存（DB非保存）
