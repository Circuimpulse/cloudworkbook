# 01. 要件定義書 (Requirements Definition)

## 1. システム概要

本システム「CloudWorkbook」は、ITエンジニア（主に応用情報技術者試験、FP3級などの資格取得を目指す学習者）向けのオンライン問題集アプリケーションである。
学習者が効率的に過去問を解き、自身の学習の進捗や正答率を管理・分析できる機能を提供する。

## 2. 目的と目標

- **目的**: 資格試験の過去問演習をスマートフォンやPCから手軽に行えるようにし、学習者の合格をサポートする。
- **目標**:
  - 隙間時間を活用しやすい「小分けの試験セクション（例: 7問1セット）」での学習体験の提供。
  - 本番を想定した「模擬テスト（50問）」機能による実力測定。
  - 学習履歴や正答率の可視化によるモチベーション維持と弱点克服の支援。

## 3. ターゲットユーザー

- 応用情報技術者試験の受験予定者（午前・午後）
- FP3級の受験予定者（学科・実技）
- その他、IT分野の知識を体系的に身につけたいエンジニア・学生

## 4. 機能要件

### 4.1. ユーザー機能

1. **認証・ユーザー管理**
   - 新規登録、ログイン、ログアウト機能（Clerkを利用）
   - ゲストユーザー向けのお試し機能（必要に応じて）
2. **学習機能（一問一答 / セクション学習）**
   - 試験別に分類されたセクション（複数問で1セット）の選択・実施
   - 4択問題などの解答機能と即時フィードバック（正解・解説の表示）
   - 中断・再開機能
3. **模擬テスト機能**
   - 全範囲からランダムに規定問題数（例:50問）を出題
   - 制限時間のタイマー表示（将来拡張）
   - 終了後のスコア算出と結果表示
4. **学習履歴・分析機能**
   - ダッシュボードでの全体進捗、正答率、受験回数の確認
   - セクション別の達成度（未着手、着手済み、マスター済み）の確認
   - 直近の模擬テスト結果の履歴表示
   - お気に入り（ブックマーク）問題の保存・復習機能
   - 間違えた問題のみを復習する機能

### 4.2. システム・管理者機能

1. **問題データ管理**
   - CSV等からの過去問データのインポート・更新（バッチ処理・スクリプト）
   - 問題、選択肢、解説、画像データの管理

## 5. 非機能要件

### 5.1. パフォーマンス・可用性

- Cloudflare Workers/Pages などのエッジコンピューティング環境を利用し、高速なレスポンスを実現する。
- データベースはエッジから低遅延でアクセス可能な Cloudflare D1（SQLite）または Turso などを利用する。

### 5.2. デバイス・UI/UX

- レスポンシブデザイン対応（スマートフォン、タブレット、PC）
- モダンで直感的なUI（グラスモーフィズム、適切なアニメーション、視認性の高いタイポグラフィ）

### 5.3. セキュリティ

- ユーザーのパスワードなどの機密情報は自社DBに持たず、外部の認証プロバイダ（Clerk）に委譲する。
- 適切なCORS設定、APIのバリデーション（Zodなど）を実施する。
