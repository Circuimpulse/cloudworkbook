# TopScreen

## 対象ソースファイル

`src/frontend/screens/TopScreen.tsx`

## 責務・概要

アプリケーションの最初に表示されるトップ画面。登録済みの過去問一覧を表示し、各試験区分の詳細ページへの導線を提供する。

## Props

| Prop    | 型       | 必須 | 説明                   |
| ------- | -------- | ---- | ---------------------- |
| `exams` | `Exam[]` | ✅   | 表示する試験区分の配列 |

## 画面構成

1. **ブランドセクション** — Cloudアイコン + アプリ名
2. **説明セクション** — アプリの概要説明テキスト（`APP_TEXTS.top` から取得）
3. **過去問一覧** — `exams` をグリッド表示（1列 / md:2列）。各カードは `CheckCircle2` アイコン + タイトル + 説明文。クリックで `/exams/{id}` へ遷移

## 使用コンポーネント

- `PageContainer` — ページ共通レイアウト
- `Cloud`, `CheckCircle2` — lucide-react アイコン

## ルーティング

- **表示パス**: `/`（`src/app/page.tsx`）
- **遷移先**: `/exams/{examId}`
