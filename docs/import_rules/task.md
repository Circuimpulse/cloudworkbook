# 試験データインポートルール

## セクション構成ルール

### 選択式試験（FP / 応用情報午前）

| ルール | 内容 |
|:---|:---|
| 問題数/セクション | **5問ずつ**に分割（端数OK） |
| セクションタイトル | 通し番号 `#1`, `#2`, ... `#N`（年度でリセットしない） |
| セクション表示順 | `1, 2, 3, ...` の連番（titleと一致） |
| セクション説明 | `年度ラベル 問X〜Y` 形式（例: `令和6年度 秋期 問1〜5`, `2024年9月 問6〜10`） |

### 記述式試験（応用情報午後 / IPA高度試験）

| ルール | 内容 |
|:---|:---|
| セクション単位 | **1大問＝1セクション**（問1, 問2, ... ごとにセクション化） |
| セクションタイトル | 通し番号 `#1`, `#2`, ... `#N` |
| セクション説明 | `年度ラベル 午後/午後Ⅰ/午後Ⅱ 問N [分野名]` 形式 |
| 応用情報午後の分野名 | 問番号に対して固定（下表参照） |

#### 応用情報午後 分野マッピング

| 問 | 分野 | 選択 |
|:---|:---|:---|
| 問1 | 情報セキュリティ | 必須 |

| 問7 | 組込みシステム開発 | 4問選択 |
| 問8 | 情報システム開発 | 4問選択 |
| 問9 | プロジェクトマネジメント | 4問選択 |

### 正解データの情報源
- **IPA公式模範解答**（`解答.md`）を正とする
- 解説.md の正解表記はAI生成のため不一致の可能性あり
