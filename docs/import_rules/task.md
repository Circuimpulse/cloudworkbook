# 試験データインポートルール

## セクション構成ルール

全試験で統一するセクション構成ルールです。

### 1. 問題数/セクション
- **5問ずつ**に分割する（全試験共通）
- 端数がある場合は最後のセクションが5問未満になってもOK

### 2. セクションタイトル (`title`)
- **通し番号** `#1`, `#2`, `#3`, ... `#N` を使用
- 年度でリセットしない（全期通しての連番）

### 3. セクション表示順 (`order`)
- `1, 2, 3, ...` の連番
- `title` の番号と一致させる（`#1` → order=1）

### 4. セクション説明 (`description`)
- 年度ラベル + 問題範囲の形式
- 例: `令和6年度 秋期 問1〜5`, `2024年9月 問6〜10`
- 年度ラベルがない試験は `問1〜5` のみでもOK

### 5. 正解データの情報源
- **IPA公式模範解答**（`解答.md`）を正とする
- 解説.md の正解表記はAI生成のため不一致の可能性あり
