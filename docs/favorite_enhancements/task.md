# お気に入り機能の強化

## 概要

お気に入り機能を拡張し、3段階のレベルを設定できるようにします。
また、学習履歴画面でのフィルタリング機能を追加します。

## 目的

ユーザーが間違えた問題やお気に入りの問題をより細かく管理し、効率的に復習できるようにするため。

## 要件

1. **お気に入りレベルの実装**
   - お気に入りを「★1」「★2」「★3」の3段階で登録できるようにする。
   - データベーススキーマを変更し、各レベルのフラグを追加する。

2. **UIの改善**
   - 問題解答画面（`StudySessionScreen`）に3段階のお気に入りボタンを配置する。
   - 学習履歴画面（`LearningHistoryScreen`）のリスト表示にお気に入りボタンを追加し、その場で状態を変更できるようにする。

3. **フィルタリング機能**
   - 学習履歴画面のお気に入りタブで、表示するレベルを選択可能にする（チェックボックス）。
   - 複数のレベルを選択した場合の条件（OR/AND）を指定できるようにする。

4. **既存機能との整合性**
   - 「間違えた問題」タブと「お気に入り」タブの切り替えを維持する。
   - 既存のお気に入りデータ（レベルなし）はレベル1として扱うなどの互換性を考慮する。
